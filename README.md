# Python Team Calculator

Welcome to **Sefin's Team Calculator**! 🎉  
This is a simple calculator built in Python by a team of 4 members. It can perform basic arithmetic operations: addition, subtraction, multiplication, and division.  

---

## **Team Members & Roles**

| Member | Responsibility |
|--------|----------------|
| Person 1 | User input & validation |
| Person 2 | Calculation logic |
| Person 3 | Output & User Interface |
| Person 4 | Integration & Testing |

---

## **Features**

- Add, subtract, multiply, divide
- Input validation (numbers only)
- Division by zero check
- Repeat calculations without restarting
- Easy to extend for GUI or more operations
